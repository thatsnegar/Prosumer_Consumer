
Code Execution Manual 

1. Environment Setup:

 To run the simulation, the following software environment is required 

    1.1 Python Version : 3.9 or later 

    1.2 Required Libraries : install the dependencies using 
        - pip install numpy matplotlib
    
    1.3 File Structure :

        simulation.ipynb (main simulation loop)
        Agent.py
        BlockChain.py
        Load.py
        Market.py
        Price_Forecast.py
        PV_Generation.py
        Regulator.py
    
2. How to run the code :
    2.Running the simulation : in the Jupyter notebook "Simulation.ipynb" run all the blocks or run each block separately
